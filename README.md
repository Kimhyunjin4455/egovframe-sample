
# 전자정부 프레임워크 기반 Spring Boot ToDo 프로젝트

## 📌 목표
전자정부 프레임워크(eGovFrame) 4.1.0과 Spring Boot 2.7.0 기반으로 ToDo 웹 애플리케이션 환경을 구축합니다.

- Spring Boot 2.7.0
- 전자정부프레임워크 4.1.0
- MyBatis
- WAR 배포
- Log4j
- Maven (pom.xml)

## 💡 주요 기능

- `real`/`staging`/`develop`/`local` 로 환경 분리
- 환경별 로깅 셋팅 
- 할 일(ToDo) 등록, 조회, 수정, 삭제 API 구현
