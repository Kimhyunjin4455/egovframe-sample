
# 전자정부 프레임워크 기반 Spring Boot ToDo 프로젝트

## 📌 목표
전자정부 프레임워크(eGovFrame) 4.1.0과 Spring Boot 2.7.0 기반으로 ToDo 웹 애플리케이션 환경을 구축합니다.

- Spring Boot 2.7.0
- 전자정부프레임워크 4.1.0
- MyBatis
- WAR 배포
- Log4j
- Maven (pom.xml)
